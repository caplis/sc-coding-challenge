SparkCentral Coding Challenge
-----------------------------

# Setup
### Install Node
```
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.32.0/install.sh | bash
$ nvm install 6.6.0
```

### Clone Repo
```
$ https://github.com/caplis/sc-coding-challenge.git
```

### Run
```
$ cd sc-coding-challenge
$ node ./index "3 * 2 + 2 * 3"`
```
